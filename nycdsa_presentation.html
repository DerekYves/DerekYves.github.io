My Teaching (R) Philosophy

  • Develop a solid grasp of basic R:
    • Base R packages and functions
    • Primary data types
    • Generalized vs specific code
    • Intelligent use of *apply and control structures
  • Use examples/data that are substantively meaningful
  • Help people learn R by trying it (coding)
    • And help them develop internal cues to signal when deeper understanding is needed.

Lets use this function to calculate some distances:

# First, save out your locations:
college <- coords[1, ]
alaska  <- coords[2, ]
woods   <- coords[3, ]
small   <- coords[4, ]

and apply coffeetime:

coffeetime(college)
## [1] 1.094508
coffeetime(alaska)
## [1] 0.6069623
coffeetime(woods)
## [1] 12.85696
coffeetime(small)
## [1] 0.8492557

And, to prove R is (also) a big calculator:

how_far <- (coffeetime(college) + 
            coffeetime(alaska) +
            coffeetime(woods) + 
            coffeetime(small)  ) / 4 *.62 

cat("The mean distance is:", round(how_far, digits=1), "miles.")
## The mean distance is: 2.4 miles.
if(how_far<5) message("You seem to like Starbucks!")
## You seem to like Starbucks!
